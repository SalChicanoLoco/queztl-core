HOW TO USE
--------------------------------------------------------------------------------

1. Backend is running on port 8000 ‚úÖ

2. Upload your MAG survey:
   curl -X POST http://localhost:8000/api/mining/mag-survey \
     -F "file=@survey.csv" \
     -F "file_format=csv"

3. Get drill targets:
   curl -X POST http://localhost:8000/api/mining/target-drills \
     -H "Content-Type: application/json" \
     -d '{"magnetic_data": [...], "locations": [...], "top_n": 10}'

4. Calculate costs:
   curl "http://localhost:8000/api/mining/survey-cost?area_km2=10"
