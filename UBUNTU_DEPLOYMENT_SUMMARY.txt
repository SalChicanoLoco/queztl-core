If SSH fails, try:

1. Test basic SSH:
   $ ssh ubuntu@<vm-ip> 'echo connected'

2. Add verbose output:
   $ ssh -v ubuntu@<vm-ip> 'echo test'

3. Check SSH key permissions:
   $ chmod 600 ~/.ssh/your-key.pem

4. Add key to SSH agent:
   $ ssh-add ~/.ssh/your-key.pem

5. Test with explicit key:
   $ ssh -i ~/.ssh/your-key.pem ubuntu@<vm-ip>

The deploy script uses standard SSH, so if manual SSH works,
the script will work.
